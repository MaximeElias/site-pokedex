Luxray


Caractéristiques

  • Nom : Luxray
  • Taille : 1.4 m
  • Poids : 42.0 kg
  • Catégorie : Pokémon Éclair
  • Type : Électrik

Description

Luxray est un Pokémon de type Électrik. Il est connu pour son allure majestueuse et son attitude protectrice. Avec la capacité de voir à travers les objets, Luxray est un chasseur efficace, utilisant ses compétences électriques pour immobiliser ses proies.

Évolutions

Luxray est l'évolution de Luxio, qui évolue à partir de Lixy.
