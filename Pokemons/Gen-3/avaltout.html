Avaltout


Caractéristiques

  • Nom : Avaltout
  • Taille : 1.2 m
  • Poids : 45.0 kg
  • Catégorie : Pokémon Boue
  • Type : Poison/Sol

Description

Avaltout est un Pokémon de type Poison et Sol. Il est connu pour sa capacité à absorber les déchets et à les transformer en boue. Ses attaques peuvent être dévastatrices pour les adversaires.

Évolutions

Avaltout évolue à partir de Gloupti à partir du niveau 30.
