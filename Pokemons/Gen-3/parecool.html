Parecool


Caractéristiques

  • Nom : Parecool
  • Taille : 0.9 m
  • Poids : 6.5 kg
  • Catégorie : Pokémon Glacial
  • Type : Glace

Description

Parecool est un Pokémon de type Glace. Il est connu pour sa capacité à se fondre dans les paysages hivernaux. Sa fourrure épaisse lui permet de résister aux températures les plus froides.

Évolutions

Parecool évolue en Vigoroth puis en Monaflemit.
