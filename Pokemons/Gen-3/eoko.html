Eoko


Caractéristiques

  • Nom : Eoko
  • Taille : 0.8 m
  • Poids : 12.0 kg
  • Catégorie : Pokémon Énergie
  • Type : Électrik/Psy

Description

Eoko est un Pokémon de type Électrik et Psy. Il est connu pour sa capacité à générer des champs électromagnétiques, ce qui lui permet d'interagir avec les appareils électroniques. Eoko est un Pokémon curieux et intelligent.

Évolutions

Eoko évolue à partir de Korillon.
