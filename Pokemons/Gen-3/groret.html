Groret


Caractéristiques

  • Nom : Groret
  • Taille : 0.9 m
  • Poids : 71.5 kg
  • Catégorie : Pokémon Magouilleur
  • Type : Psy

Description

Groret est un Pokémon de type Psy. En utilisant les perles noires incrustées sur son corps, il est capable de manipuler les ondes psychiques pour déstabiliser ses adversaires. Son énergie mentale est fortement amplifiée à mesure qu'il grandit.

Évolutions

Groret évolue à partir de Spoink au niveau 32.
