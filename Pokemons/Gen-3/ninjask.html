Ninjask


Caractéristiques

  • Nom : Ninjask
  • Taille : 1.0 m
  • Poids : 12.0 kg
  • Catégorie : Pokémon Ninja
  • Type : Insecte / Vol

Description

Ninjask est un Pokémon de type Insecte / Vol. Il est réputé pour sa vitesse incroyable, capable d'esquiver presque tous les attaques. Grâce à sa capacité à se fondre dans l'ombre, il devient un maître du camouflage.

Évolutions

Ninjask évolue à partir de Ningale et n'a pas d'évolution.
