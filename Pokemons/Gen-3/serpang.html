Serpang


Caractéristiques

  • Nom : Serpang
  • Taille : 1.2 m
  • Poids : 25.0 kg
  • Catégorie : Pokémon Serpent
  • Type : Eau/Poison

Description

Serpang est un Pokémon de type Eau et Poison. Il est connu pour sa capacité à nager à grande vitesse et à se camoufler dans l'eau pour surprendre ses proies. Serpang est également capable de produire un venin puissant.

Évolutions

Serpang évolue à partir de Coquiperl.
