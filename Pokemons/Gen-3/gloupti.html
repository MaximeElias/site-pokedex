Gloupti


Caractéristiques

  • Nom : Gloupti
  • Taille : 0.8 m
  • Poids : 70.5 kg
  • Catégorie : Pokémon Résidu
  • Type : Poison

Description

Gloupti est un Pokémon de type Poison. Il se nourrit de déchets et de restes alimentaires. Gloupti a la capacité de se dilater pour absorber davantage de matière, ce qui lui permet de survivre dans des environnements pollués.

Évolutions

Gloupti évolue en Avaltout.
