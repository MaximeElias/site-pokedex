Négapi


Caractéristiques

  • Nom : Négapi
  • Taille : 0.4 m
  • Poids : 4.2 kg
  • Catégorie : Pokémon Acclameur
  • Type : Électrik

Description

Négapi est un Pokémon de type Électrik. Contrairement à son partenaire Posipi, il est souvent vu comme plus pessimiste. Cependant, il est tout aussi efficace pour encourager ses alliés durant les combats, en dépit de son attitude réservée.

Évolutions

Négapi n'a pas d'évolution.
