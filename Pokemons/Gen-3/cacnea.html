Cacnea


Caractéristiques

  • Nom : Cacnea
  • Taille : 0.4 m
  • Poids : 51.3 kg
  • Catégorie : Pokémon Cactus
  • Type : Plante

Description

Cacnea est un Pokémon de type Plante. Il vit dans les déserts arides, attendant patiemment ses proies. Pour se nourrir, il utilise ses épines acérées pour attaquer et immobiliser les Pokémon qui s'aventurent trop près.

Évolutions

Cacnea évolue en Cacturne au niveau 32.
