Balignon


Caractéristiques

  • Nom : Balignon
  • Taille : 0.5 m
  • Poids : 1.2 kg
  • Catégorie : Pokémon Spore
  • Type : Plante

Description

Balignon est un Pokémon de type Plante. Il est connu pour sa forme ronde et ses couleurs vives. Balignon aime se cacher dans les herbes hautes et se nourrit de nutriments du sol.

Évolutions

Balignon évolue en Chapignon.
