Barbicha


Caractéristiques

  • Nom : Barbicha
  • Taille : 0.9 m
  • Poids : 32.0 kg
  • Catégorie : Pokémon Récif
  • Type : Eau/Sol

Description

Barbicha est un Pokémon de type Eau et Sol. Il est connu pour sa grande résistance et sa capacité à se cacher dans les fonds marins, attendant sa proie.

Évolutions

Barbicha est l'évolution de Barloche, qui évolue à partir du niveau 30.
