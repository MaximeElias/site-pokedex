Pifeuil


Caractéristiques

  • Nom : Pifeuil
  • Taille : 1.0 m
  • Poids : 28.0 kg
  • Catégorie : Pokémon Malice
  • Type : Plante/Ténèbres

Description

Pifeuil est un Pokémon de type Plante et Ténèbres. Ce Pokémon espiègle aime semer le chaos dans les forêts en attaquant les voyageurs qui passent près de lui. Il utilise les feuilles acérées sur sa tête pour se battre.

Évolutions

Pifeuil évolue à partir de Grainipiot et évolue ensuite en Tengalice.
