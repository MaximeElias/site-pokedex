Chapignon


Caractéristiques

  • Nom : Chapignon
  • Taille : 1.2 m
  • Poids : 12.0 kg
  • Catégorie : Pokémon Champignon
  • Type : Plante, Poison

Description

Chapignon est un Pokémon de type Plante et Poison. Il est connu pour sa taille impressionnante et ses couleurs éclatantes. Chapignon se trouve souvent dans les forêts denses, où il se camoufle parmi les champignons.

Évolutions

Chapignon évolue à partir de Balignon.
