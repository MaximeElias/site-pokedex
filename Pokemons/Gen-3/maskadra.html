Maskadra


Caractéristiques

  • Nom : Maskadra
  • Taille : 1.2 m
  • Poids : 22 kg
  • Catégorie : Pokémon Papillon
  • Type : Insecte, Vol

Description

Maskadra est un Pokémon de type Insecte et Vol. Il est connu pour sa capacité à se camoufler dans son environnement, ce qui lui permet de surprendre ses proies.

Évolutions

Maskadra évolue de Arakdo.
