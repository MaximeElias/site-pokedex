Rosabyss


Caractéristiques

  • Nom : Rosabyss
  • Taille : 1.2 m
  • Poids : 23.0 kg
  • Catégorie : Pokémon Rose
  • Type : Eau

Description

Rosabyss est un Pokémon de type Eau. Il est célèbre pour sa beauté et sa grâce dans l'eau. Rosabyss utilise ses nageoires pour se déplacer avec élégance et peut hypnotiser ses adversaires avec ses mouvements.

Évolutions

Rosabyss évolue à partir de Coquiperl.
