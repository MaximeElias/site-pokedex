Tarinor


Caractéristiques

  • Nom : Tarinor
  • Taille : 1.0 m
  • Poids : 97.0 kg
  • Catégorie : Pokémon Boussole
  • Type : Roche/Acier

Description

Tarinor est un Pokémon de type Roche et Acier. Il utilise le magnétisme de son long nez pour attirer des objets métalliques. Il peut aussi détecter les directions avec précision, ce qui lui a valu le nom de Pokémon Boussole.

Évolutions

Tarinor évolue en Tarinorme lorsqu'il est exposé à une Pierre Sinnoh.
