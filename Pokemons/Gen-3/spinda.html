Spinda


Caractéristiques

  • Nom : Spinda
  • Taille : 1.1 m
  • Poids : 5.0 kg
  • Catégorie : Pokémon Panda Tache
  • Type : Normal

Description

Spinda est un Pokémon de type Normal connu pour sa démarche vacillante. Ses taches uniques sur le corps font en sorte qu'il n'existe pas deux Spinda identiques, ce qui en fait un Pokémon très particulier dans le monde du Pokédex.

Évolutions

Spinda n'a pas d'évolution.
